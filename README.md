# trust.exe

A choice-driven interactive media project built using the iSpring Suite. Part narrative experiment and part psychological thriller, **trust.exe** challenges the user’s perception of security, digital autonomy, and execution control through a highly stylized, gamified interface.

Designed to push slide-based authoring tools past standard corporate limits, this project bridges advanced logical architecture with rich, atmospheric world-building.

---

## 👁️ Core Experience Pillars

### For Designers: Diegetic UI/UX
The interface is built entirely as an in-universe, high-stakes terminal system. Every button, prompt, and transition serves the narrative atmosphere. Custom-timed assets and responsive state changes break away from traditional layouts to create an unsettling, sleek user experience.

### For Gamers: Choice-Driven Tension
Your choices dictate the outcome. The project utilizes hidden variable tracking to measure user compliance and risk-taking, altering the system state and splitting the narrative into multiple distinct endings. 

### For Professionals: Advanced Logic Architecture
* **Non-Linear State Machine:** Implements complex branching paths, conditional triggers, and state variables within a slide-based framework.
* **Engagement Architecture:** Showcases how high-frequency micro-interactions and psychological tension can be utilized to maximize user retention and deep interaction.

---

## 🛠️ Tech & Design Stack
* **Authoring Engine:** iSpring Suite (Advanced branching logic, conditional actions, and multi-tier triggers)
* **Design Pipeline:** Custom high-contrast vector UI elements and optimized typography for deep visual immersion
* **Logic Flow:** Multi-bottleneck narrative architecture with variable tracking
